✨ Features

🎯 Multiple Tasks

Pick-and-place, drawer manipulation, and tool use tasks with customizable rewards

👁️ Vision-Based

RGB input with CNN and ViT encoders for scene understanding

🧠 Flexible Policies

MLP and RNN networks with temporal reasoning capabilities

📚 Two Training Methods

Behavior cloning (imitation) and reinforcement learning (PPO/SAC)

📊 Reproducible

Deterministic seeding, batch evaluation, and JSON metrics

🔧 CI/CD Ready

GitHub Actions, Docker support, automated testing

🚀 Quick Start

Get started in 5 minutes:

1. Installation

git clone https://github.com/omargarraoui/mujoco-rl.git cd mujoco-rl bash scripts/setup.sh

2. Run Demo

python scripts/demo.py --task pick_place

3. Train Policy

# Behavior Cloning python train/train_bc.py --task pick_place --epochs 50 # Reinforcement Learning python train/train_rl.py --task pick_place --algo ppo

4. Evaluate

python eval.py \ --task pick_place \ --checkpoint checkpoints/bc_pick_place_final.pt \ --episodes 50 \ --render
